#### 一、SQL分类

* DDL—数据定义语言(CREATE，ALTER，DROP，DECLARE)
* DML—数据操纵语言(SELECT，DELETE，UPDATE，INSERT)
* DCL—数据控制语言(GRANT，REVOKE，COMMIT，ROLLBACK)
